# StarWars API Assignment
## Code the Dream Pre work

## To run the project:
- Go into the terminal
- Run command `git clone https://github.com/demenkoms/ctd-pre-work.git`
- Run command `cd ctd-pre-work` (if on Mac)
- Run the index.html file
